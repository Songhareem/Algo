
# 논리와 증명

+ 3단 논법
    + p->q 에서, p가 거짓 이면 전체는 참
    + p->q 에서, q가 참이면 전체는 참
    + p->q 에서, q가 거짓이면 전체는 거짓

+ 귀납법
    + p(1)이 참일때, p(n) -> p(n+1)이 참이라면, p(n)은 참이다 (n=모든 자연수)
    + 당구공 문제 : 당구공이 2->1 개가 될 때, 색이 하나만 남지만, 당구공 2개일 때 색이 2개인지 알 수 없음

+ 귀납법의 강한 형태
    + p(1)이 참일때, p(1) ^ p(2) ^ ... ^ p(n) -> p(n+1)이 참이면, p(n)은 참이다 (n=모든 자연수)
